Otterdeploy is a self-hosted PaaS that lets you declare your entire infrastructure in a single config file and sync it to your servers. Define services, databases, environment variables, domains, and dependencies — then deploy across multiple environments with a single command.

## Key Features

- **Declarative config** — One file defines all resources, environments, links, and variables
- **Multi-environment** — Production, staging, dev with deep inheritance
- **Real-time dashboard** — Visualize architecture, trigger deployments, monitor resources
- **Git-driven deploys** — Connect GitHub repos and deploy on push
- **Secrets management** — Infisical integration with native breakglass fallback
- **Multi-tenancy** — Organizations with role-based access control

## How It Works

```
otterdeploy.config.ts      Define your infrastructure
        |
   otterdeploy preview     See what would change
        |
   otterdeploy sync        Apply changes to your servers
```

Otterdeploy compares your desired config against the current state and generates the minimal set of create, update, and delete actions needed to reconcile them.
