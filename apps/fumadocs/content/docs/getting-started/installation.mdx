## Prerequisites

- [Bun](https://bun.sh) >= 1.3.7
- [Docker](https://docs.docker.com/get-docker/) & Docker Compose
- PostgreSQL 16 (provided via Docker)

## Clone and Install

```bash
git clone https://github.com/otterdeploy/otterdeploy.git
cd otterdeploy
bun install
```

## Environment Configuration

Copy the example environment file and edit it with your values:

```bash
cp .env.example .env
```

The key variables you need to configure:

| Variable | Description |
|---|---|
| `DATABASE_URL` | PostgreSQL connection string |
| `BETTER_AUTH_SECRET` | Auth session secret (32+ chars) |
| `BETTER_AUTH_URL` | Auth base URL |
| `CORS_ORIGIN` | Allowed CORS origin |

See `.env.example` for the full list with defaults.

## Start Infrastructure

Start PostgreSQL, Inngest (job queue), and Zero Cache (real-time sync):

```bash
bun run infra:up
```

## Push Database Schema

Apply the Drizzle schema to your local PostgreSQL:

```bash
bun run db:push
```

## Start Development

```bash
bun run dev
```

This starts all apps in parallel:
- **Web** — `http://otterdeploy.localhost:1355`
- **Server** — `http://api.otterdeploy.localhost:1355`
- **Worker** — Inngest job processor on port 3002

## Verify

Open `http://otterdeploy.localhost:1355` in your browser. You should see the login page. Create an account and you're ready to go.
