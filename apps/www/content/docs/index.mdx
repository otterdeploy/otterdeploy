Otterstack is a self-hostable deployment platform. You bring a server (or a
cluster); it gives you Git-sourced builds, a Caddy edge with automatic HTTPS,
managed databases, live logs, and built-in observability.

## What you can deploy

- **Services** — apps built from a Git repo via Railpack (Node, static SPAs, …)
- **Databases** — managed Postgres with scheduled backups
- **Edge routing** — multi-domain services behind a Caddy proxy, with
  Vercel-style deployment protection
