The Otterstack API is built on [oRPC](https://orpc.unnoq.com). The server already
emits an **OpenAPI 3.1** document from the same Zod-typed contracts that power the
dashboard.

<Callout type="info" title="Interactive reference — next step">
  This page is a placeholder. The interactive, "Send Request" API reference will
  be generated here with `fumadocs-openapi`, fed by the server's live spec at
  `/api-reference/spec.json`. See the wiring note in the repo.
</Callout>

## The pipeline

```text
oRPC contracts (packages/api)
  └─ OpenAPIHandler  (apps/server/src/index.ts)   → live spec at /api-reference/spec.json
       └─ fumadocs-openapi (apps/www)             → reference pages + playground
```

Until that's wired, the server also serves a live Scalar reference at
`/api-reference` via the oRPC `OpenAPIReferencePlugin`.
